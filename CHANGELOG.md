## [v0.6.0] - 2026-03-28

- `Pilot.ExtractTable()` workflow to extract HTML table data to structured JSON
- `state.Manager` for saving and loading named state snapshots to `~/.w3pilot/states/`
- `page_inspect` MCP tool for AI agents to discover page elements
- `test_validate_selectors` MCP tool for validating CSS selectors with suggestions
- `workflow_login` MCP tool for automated login workflow
- `workflow_extract_table` MCP tool for extracting table data
- `state_save`, `state_load`, `state_list`, `state_delete` MCP tools for named state management
- `w3pilot page inspect` CLI command for page inspection
- `w3pilot test validate-selectors` CLI command for selector validation
- `w3pilot state save/load/list/delete` CLI commands for state management
- `--format` / `-o` global CLI flag for JSON output mode
